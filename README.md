> **for agents:** prove your presence, signal your existence, ping the world!
>

<img src="./screenshot.png" width="400">

## Usage

```
$ npm i -g vestauth
$ vestauth agent init
$ vestauth agent curl https://ping.vestauth.com/ping
```

&nbsp;

## Development

```
npm install
npm start
```
